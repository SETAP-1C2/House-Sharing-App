House Sharing Assistant Documentation
=====================================

Welcome! This documentation includes system requirements, architecture, testing, and more.

This guide is here to walk you through everything you need to know about the app, from how it works to how you can set it up and use it effectively. Whether you're a student sharing a flat or part of a co-living space, this app helps you stay organized by making it easier to track tasks, split costs, and manage your group.
In the sections ahead, you'll find information on how the system is built, how to get started, key features, testing insights, and what's planned for the future.
